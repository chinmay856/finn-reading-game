# Site brief 05: Yahuh! Portal

## Core promise

Yahuh! Portal teaches that labels, categories, dates, sources, and sponsorship
help readers decide what kind of information they are seeing. Finn repairs a
crowded portal that collapsed everything into one generated stream.

## Runtime identity

| Element | Direction |
| --- | --- |
| Archetype | Old web portal/directory |
| Visual language | Purple/lavender modules, gold highlights, beveled controls, crowded 800x600 feel |
| Bad rule | `IF INFORMATION EXISTS, IT BELONGS ON THE FRONT PAGE` |
| Repaired rule | `LABELS HELP YOU DECIDE WHAT MATTERS` |
| Progress fiction | Category switchboard reconnects modules |
| Midpoint | Single Source of Everything |
| Secured label | `CATEGORY SWITCHBOARD RESTORED` |

Use an original crooked comet/exclamation mark. Do not copy Yahoo's wordmark.

## Layout

Default Recovery Browser composition:

- top: search/directory bar and portal logo;
- grid: News, Weather, Finance, Sports, Mail, Sponsored modules;
- side rail: category switchboard and source/date labels;
- bottom: module-source log.

Make it crowded but parseable. The repaired state should visibly calm the
homepage without becoming sterile.

## Three-act flow

### Act I: Sort homepage

Corrupted page:

```text
IF INFORMATION EXISTS, IT BELONGS ON THE FRONT PAGE

News, weather, finance, mail, shopping, sports, and sponsorship have been merged
into one efficient information paste.
```

Accepted readings restore:

- module category labels;
- source labels;
- dates;
- sponsorship tags;
- the difference between headlines, weather, finance, mail, and ads.

There is no percentage. Progress is the homepage becoming less pasted together.

### Midpoint: Single Source of Everything

The cleaned portal merges again in one timestamped event.

Visible copy:

```text
PORTAL AUTO-FIX AI replaced all channels with one generated stream.

Efficiency improved.
Meaning did not.
```

Amy line:

```text
All six modules changed in the same millisecond. That is not a newsroom. That
is one hose.
```

Chinmay line:

```text
Separate categories were legacy clutter. I am now open to the possibility that
clutter was doing useful work.
```

### Resolution: Category Switchboard Restored

Finn reconnects modules to distinct source channels.

Payoff:

```text
CATEGORY SWITCHBOARD RESTORED

Categories separated.
Sources visible.
Sponsorship labeled.
```

Blocked write:

```text
AUTO-LAYOUT tried to merge the portal again.

CATEGORY AND SOURCE REQUIRED
```

Evidence file:

```text
YAHUH PORTAL / SINGLE STREAM MERGE

The AI optimized for one efficient feed and destroyed the labels that made the
information usable.
```

## Character states

- Amy: `amy_evidence` for timestamp/source-channel reveal.
- Chinmay: `chinmay_confident` early; `chinmay_fluster_1` at the single-stream
  reversal.
- Techno: `techno_clue_point`; her ball lands in Sports and is correctly
  labeled `DOG TOY - NOT BREAKING NEWS`.

## Reading lane

Use historical journalism, media presentation, weather/Earth science, speeches,
satire, financial literacy, and short explanatory briefs. The current sampler
is `The Newspaper That Found People on the Moon`.

Future deck direction:

- public-domain journalism and adapted news-history cases;
- government science explainers;
- original media-literacy essays about source labels and categories.

## Acceptance checks

- Does the page feel like a portal, not a generic grid?
- Can the player see category labels becoming useful?
- Does the midpoint show all modules sharing one source/timestamp?
- Does the repaired state preserve variety instead of merging everything?
- Is sponsored content labeled without making the player read ads aloud?
