# Internet Recovery 98 production character sheets

These are builder-ready wrapper assets for the first Internet Recovery 98
prototype. They are production character references, not final cropped runtime
sprites. The builder may crop individual panels from each sheet while preserving
the panel order and asset IDs below.

The sheets deliberately contain no embedded text labels. Use this manifest as
the source of truth for IDs.

## Sheets

- `amy-production-portraits.png` - production Amy support portraits and poses.
- `chinmay-production-portraits.png` - production Chinmay portrait states.
- `techno-production-sprites.png` - production Techno desktop-pet sprites.

shown through messier hair, loosened clothes, and frazzled expressions, never
anger or threat.

## Use boundaries

- These assets belong to the Internet Recovery 98 wrapper only.
- Do not place character names, moods, or story state in the Reading Engine.
- Private reference photos are not committed and must not be recreated as
  photoreal panels.
- Character reactions must not cover passage text, microphone controls, live
  highlighting, or results.
