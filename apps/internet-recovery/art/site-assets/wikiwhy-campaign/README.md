# WikiWhy campaign-state assets

These assets add campaign-only state to the implemented WikiWhy reader. They do
not replace the passage, Reading Companion, microphone UI, score, or
comprehension controls.

| Asset ID | File | View box | Use |
| --- | --- | --- | --- |
| `wikiwhy.evidence.route-fragment-01` | `evidence-route-fragment-01.svg` | `720 420` | Case File evidence slot 1 and access-route fragment. |
| `wikiwhy.secured.seal` | `wikiwhy-secured-seal.svg` | `128 128` | Permanent secured indicator on site window, shortcut, and Recovery Map tile. |

The evidence asset contains meaningful text. Repeat its fields in accessible DOM
content; the SVG cannot be the sole representation.

The corrupted article's toast art, if retained, is decorative and is not the
WikiWhy evidence receipt.
