# Site preview crops

These optimized JPEG crops come directly from the preferred `*-rogue-ai-*`
concept boards under `art/concepts/`. They let the Recovery Map and design-
preview windows reuse the approved visual source without loading eleven
multi-megabyte review boards into the live prototype.

Each crop uses the first-act region of its source board, resized to 640 by 480
at JPEG quality 84. They remain wrapper-owned presentation only. A crop does not
mean that a site's passage, mechanics, story state, or production page is
implemented or approved. WikiWhy is still the only playable site.
