# Site preview crops

These optimized JPEG crops come directly from the canonical builder-ready
`*-rogue-ai-campaign.png` concept boards under `art/concepts/`. They let the
Recovery Map and design-preview windows reuse the approved design reference
without loading eleven multi-megabyte review boards into the live prototype.

Each crop uses the 512 by 384 first-act region at offset `0,40`, resized to 640
by 480 at JPEG quality 84. MyCorner, Search-ish, Amaze-On, Spotty-Fi, and
MapGuess were regenerated after their corrected boards were promoted; files
with `-superseded` in the source name are never builder inputs. The crops remain
wrapper-owned presentation only. A crop does not mean that a site's passage,
mechanics, story state, or production page is implemented. WikiWhy is still the
only playable site.
