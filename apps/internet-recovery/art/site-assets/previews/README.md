# Site preview crops

These optimized JPEG crops come from generated `*-rogue-ai-campaign.png`
storyboard references under `art/concepts/`. They let the Recovery Map and
design-preview windows identify unfinished sites without loading eleven
multi-megabyte review boards. They are preview-only history, not production
identity, copy, chrome, or state input.

Each crop uses the 512 by 384 first-act region at offset `0,40`, resized to 640
by 480 at JPEG quality 84. MyCorner, Search-ish, Amaze-On, Spotty-Fi, and
MapGuess were regenerated after their corrected storyboards were selected;
files with `-superseded` in the source name are never preview sources. The crops
remain wrapper-owned presentation only. A crop does not mean that a site's passage,
mechanics, story state, or production page is implemented. WikiWhy is still the
only playable site.
